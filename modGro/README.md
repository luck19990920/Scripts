`modGro`是用于修改`GROMACS`中的`gro`文件的工具，采用C++编写。`modGro.exe`是编译后得到的Windows可执行文件，`modGro`是编译后得到的Linux可执行文件 <br>

使用方法详见<a href="https://blog.99092019.xyz/modGro_usage.html">modGro的使用方法</a>
